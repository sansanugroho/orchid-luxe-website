Semua perubahan kamu akan dimasukkan ke dalam file index.html baru. Upload file ini ke GitHub, dan Netlify akan otomatis update dalam 30 detik.

  1. 1
    Klik "Download index.html" di bawah
  2. 2
    Buka github.com → repository kamu
  3. 3
    Tap file index.html lama → tap ✏️ → scroll bawah → "Delete file"
  4. 4
    Upload file baru yang baru kamu download → Commit
  5. 5
    Tunggu 30 detik → buka orchidluxebouquets.netlify.app ✨
